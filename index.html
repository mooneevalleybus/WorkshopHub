    Export a full backup of all workshop data (fleet, tasks, parts, job cards, defects, accidents, companies, pre-trip history, Jaltest details) to a JSON file. If SharePoint becomes unavailable, this file can be used to restore your data.

    Importing will merge the backup into SharePoint — existing items with matching IDs will be updated, and new items will be created. This may take a few minutes for large backups.
